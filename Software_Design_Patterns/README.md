# Design Patterns in C++ — Reference Repository

## Overview

This repository is a **structured, C++-focused reference implementation** of the classic **Gang of Four (GoF) Design Patterns**.  
Each design pattern lives in its own folder and contains:

- A standalone C++ implementation (`.cpp`)
- A detailed `README.md` explaining intent, architecture, advantages, and usage

This central README serves as a **navigation hub**.

---

## Repository Structure

Each pattern directory follows this structure:
pattern.cpp
README.md
